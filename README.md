# Man Hunt Backend
This program is a part of my game project Man Hunt and written completely in Java. It is just a simple implementation of UDP protocol backend which provides multiplayer feature for my game.

# Features
- Manage player connection (Connect/Disconnect)
- Room create/room leave
- Communicate among clients in room
- Game logic execution on server side
